# You Okay? - Minimalist Glassmorphism Note Engine

You Okay? is a lightweight, privacy-focused Progressive Web App (PWA) designed for seamless note-taking, folder organization, and local-first document management. Powered by vanilla JavaScript, IndexedDB, and modern CSS glassmorphism, it delivers a clean, responsive, and secure note-writing experience across desktop and mobile devices.

## Live Demo
Access the hosted application here: **[https://agnezgrace.github.io/YouOkay/]**

## Features
*   **Progressive Web App (PWA)**: Full offline support enabled by a dedicated Service Worker and Web App Manifest.
*   **IndexedDB Local Storage**: Local-first architecture ensuring all notes and custom folders are stored securely inside your browser with instant data retrieval.
*   **Folder & Filter Management**: Organize notes into custom folders, filter by pinned status, or view all notes in a unified sidebar.
*   **Real-Time Search & Sorting**: Instant search across titles and content, alongside flexible sorting options (Newest, Oldest, Title A-Z).
*   **Auto-Save & Live Statistics**: Automatic draft saving with a visual status indicator, plus real-time word and character counters.
*   **Import & Export Capability**: Seamlessly import `.txt` or `.json` note files and export active notes to plain text files.
*   **Responsive Glassmorphism UI**: Modern ambient visual design with collapsible sidebars, custom dropdowns, confirmation modals, and toast notifications optimized for desktop and mobile screens.
*   **Privacy & Client-Side Focused**: Zero tracking, zero telemetry, and no external servers required.
